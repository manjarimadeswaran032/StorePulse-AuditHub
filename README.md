# StorePulse Audit Hub

**StorePulse Audit Hub** is a Departmental Store Audit Dashboard developed to analyze store operations, detect irregularities, and improve inventory, billing, and compliance using data analytics.

---

## 📊 Project Overview

Departmental stores handle large volumes of daily transactions including sales, billing, inventory updates, and staff activities.

Errors or inconsistencies can lead to:

* Financial losses
* Inventory mismatches
* Unauthorized discounts
* Compliance issues

This dashboard helps identify such issues using data-driven auditing techniques.

---

## 🚀 Features

* Upload store transaction datasets (CSV/Excel)
* Detect billing errors and duplicate transactions
* Identify unauthorized discounts
* Analyze stock discrepancies
* Visualize trends using charts
* Generate audit findings and reports

---

## 📊 Dashboard Modules

### 1. Overview

* Total audit records
* Stock discrepancies
* Billing errors
* Compliance score

### 2. Transaction Logs

* Employee activity tracking
* Sales, refunds, discounts
* Suspicious activity detection

### 3. Audit Findings

* Unauthorized discounts
* Billing mismatches
* Missing inventory
* Expired products

### 4. Trends & Patterns

* Sales vs inventory trends
* Peak suspicious activity times
* Department-wise risk analysis

### 5. Risk Analysis Report

* Risk levels (Low / Medium / High)
* Key issues identified
* Recommendations for improvement

---

## ⚠️ Key Audit Findings

* Frequent stock mismatches detected
* Unauthorized discounts applied by staff
* Duplicate billing entries found
* Expired items still available for sale
